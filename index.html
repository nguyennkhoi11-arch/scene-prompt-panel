Cách sử dụng
3 BƯỚC LÀ XONG

Không cần cài đặt phức tạp. Mở extension, paste dữ liệu, bắt đầu làm việc ngay.

01

Paste dữ liệu

Mở file .txt script, Ctrl+A → Ctrl+C → paste vào panel. Extension tự nhận dạng Markdown hoặc plain text.

02

Chọn scene

Duyệt qua danh sách Acts và Scenes ở sidebar. Tìm kiếm nhanh. Điều hướng bằng phím mũi tên ← →.

03

Copy & dùng

Nhấn COPY trên từng prompt — START FRAME, END FRAME, VIDEO PROMPT. Paste vào bất kỳ AI nào.

Tính năng
MỌI THỨ BẠN CẦN
📂

Auto-detect format

Tự động nhận cả 2 format: Markdown (#, ##, ###) và plain text. Không cần cấu hình gì.

⚡

Copy 1 click

START FRAME, END FRAME, VIDEO PROMPT — mỗi section có nút COPY riêng. Không cần select text.

🔍

Tìm kiếm nhanh

Search scene theo tên. Điều hướng bằng phím mũi tên. Không bao giờ mất vị trí đang làm việc.

🎨

Tùy biến tự do

Đặt tên section thoải mái — IMAGE START, KHUNG ĐẦU, OPENING SHOT... Extension đọc được hết.

🔒

An toàn 100%

Không can thiệp vào Flow hay bất kỳ website nào. Chỉ dùng clipboard. Không thu thập dữ liệu.
